			Handling the exceptions in Python
		============================================
=>Handling the exceptions in Python is nothing but converting Technical error messages into user-friendly error messages. To do this process, as part exception handling concept , we have 5 keywods. They are

			1) try
			2) except 
			3) else
			4) finally
			5) raise
--------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------

			try:
			       -------------------------------------------------------------
